# 🍰 New Poona Bakery Clone

A professional, fully responsive web clone of the iconic **New Poona Bakery (NPB)**. This project focuses on replicating the premium UI/UX of the original site, including custom animations and brand-specific design elements.

## 🚀 Live Demo


## ✨ Key Features

- **Custom Navbar:** Centered logo logic with professional dividers and hover effects.
- **Hero Slider:** A 4-slide Bootstrap carousel with synchronized `fadeInUp` text animations.
- **Premium UI Details:** - **Jali Texture:** A custom CSS grid overlay on slider images for that authentic NPB look.
  - **Decorative Overlap:** Rolling Pin (Belan) PNGs used as seamless section dividers.
- **Product Grid:** 8-card responsive layout with hover-triggered product names and opacity effects.
- **Testimonial Slider:** Featuring custom-styled indicators (radio dots) and customer reviews (Subodh Bhave, etc.).
- **Interactive Map Overlay:** Contact Us form floating over a live Google Maps integration.
- **Modern Footer:** 4-column dark theme footer with social icons and heritage branding.

## 🛠️ Tech Stack

- **HTML5 & CSS3** (Custom Styling & Keyframe Animations)
- **Bootstrap 5** (Grid System & Components)
- **AOS (Animate On Scroll)** (For smooth scroll-triggered entry)
- **FontAwesome 6** (Vector Icons)
- **Google Fonts** (Poppins & Playfair Display)
